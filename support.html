Getting started

Vibeit is a full Python IDE that runs on your device. Create a notebook or script, write Python, and press Run — code executes in an on-device kernel with support for notebooks, matplotlib, NumPy, pandas, PyTorch, and more. No internet connection is required to run your code.

Frequently asked questions

What's included in the free version vs. a subscription?

The core on-device Python IDE is available with the Standard plan. Pro adds remote SSH / SFTP servers and terminal, GitHub clone / push / pull & pull requests, and cloud AI providers (Claude, Codex, and custom). Every plan starts with a 7-day free trial.

How do I start the free trial or subscribe?

Open Settings → Account in the app, choose a plan (monthly, quarterly, or yearly), and tap Start Free Trial. Payment is handled by Apple through your Apple Account.

How do I manage or cancel my subscription?

Subscriptions are managed by Apple. On your device, open the Settings app → tap your name → Subscriptions → Vibeit, where you can change plan or cancel. If you cancel during the free trial before it ends, you won't be charged. You can also manage subscriptions here.

How do I restore a purchase?

On a new device or after reinstalling, open Settings → Account in the app and tap Restore Purchases. Make sure you're signed in with the same Apple Account you used to subscribe.

How do I connect a remote server or GitHub? (Pro)

Go to Settings → Remote to add an SSH/SFTP host, or Settings → Source Control to connect GitHub. Your credentials are stored securely in the device Keychain and are never sent to us.

How do I use my own AI provider? (Pro)

Open Settings → AI and add your provider and API key (for example Claude or Codex). Your key stays on your device and is used only to talk directly to that provider.

Where can I install additional Python packages?

Use the in-app package catalog to install supported pure-Python and prebuilt native packages. Availability depends on what can run on iOS/iPadOS.
